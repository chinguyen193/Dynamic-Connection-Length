This folder provided all file to run the LISFLOOD-FP model for the flood event in 2013 at Narran Park, Australia.
Detail to run the model can be found in 'LISFLOOD-FP user manual.pdf" in the folder.
For example to run the model in this folder, open the command window or Powershell window, direct to the folder and type:
.\\lisflood.exe -v Narran.par
